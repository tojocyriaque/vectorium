# Slide 1: Title Slide
- Project Name: Physics Animator
- Tagline: AI-powered physics animation & narration engine

# Slide 2: Problem Statement
- Difficulty of learning physics concepts
- Lack of intuitive visualization tools
- Static diagrams are not enough
- Students struggle with motion understanding

# Slide 3: Solution Overview
- AI-generated physics animations from text
- Real-time simulation rendering
- Educational narration system
- Vector-based physics visualization

# Slide 4: System Architecture
- FastAPI backend
- LLM providers (Groq + Gemini)
- JSON-based animation schema
- Pygame rendering engine

# Slide 5: AI Physics Engine
- Natural language to structured physics events
- Keyframes for motion simulation
- Physics rules embedded in prompts
- Validation layer for safety and consistency

# Slide 6: Physics Rendering System
- Object-based animation model
- Keyframe interpolation
- Collision and gravity simulation
- Canvas-based rendering (600x400 system)

# Slide 7: Vector Physics System (NEW)

# Slide 8: Physics Narration Engine (NEW)

- Event-driven explanation timeline
